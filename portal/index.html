Setup Instructions

  1. Install dependencies: pip install -r requirements.txt
  2. Sign up at twilio.com (free trial = 75 mins of calls, no credit card needed)
  3. Set environment variables:
    export TWILIO_ACCOUNT_SID=your_sid
    export TWILIO_AUTH_TOKEN=your_token
    export TWILIO_PHONE_NUMBER=+1234567890
    export BASE_URL=https://your-ngrok-url.ngrok.io
  4. Install ngrok and run: ngrok http 5000
  5. Set BASE_URL to the ngrok HTTPS URL
  6. Start the server: python app.py
  7. Open this file in your browser
